Fallback Jobs
=============

The :class:`~compute_horde_sdk.v1.fallback.FallbackJobSpec` class is used to define a fallback job specification.
An instance of this class contains all the parameters of a job to be sent to fallback cloud.

The :class:`~compute_horde_sdk.v1.fallback.FallbackJob` class represents a fallback job that runs on a fallback cloud.
This class includes methods checking the status and results of the job.
